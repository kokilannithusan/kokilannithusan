### 📂 Featured Projects:

#### 🏨 [Hotel Management System](https://github.com/kokilannithusan) (Active Development- Group)
*A full-stack application designed to automate hotel operations, focusing on secure access and efficient resource management.*
- **Key Implementation:**
    - **Secure Authentication:** Developed a login system using **JWT (JSON Web Tokens)** to provide secure, stateless authentication.
    - **RESTful APIs:** Built backend endpoints to validate credentials and manage user sessions.
    - **Data Management:** Implemented core **CRUD operations** for managing hotel data.
    - **Future Roadmap:** Implementing a **User Privilege System** (Role-Based Access Control) to handle different user permissions.
- **Tech Stack:** Spring Boot, Spring Security, React, TypeScript, MySQL.

- **Key Features:** - **Multi-Role Ecosystem:** Specialized dashboards for Farmers, Delivery Personnel, Clerks, and Branch Managers.
    - **District-Level Management:** Administrative hierarchy with dedicated managers for every Sri Lankan district.
- **Tech Stack:**  JavaScript, PHP, MySQL, HTML, CSS, Bootstrap.
